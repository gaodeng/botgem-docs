# OpenAI Configuration Guide

This guide explains how to set up and configure OpenAI as an AI service provider in BotGem.

## Getting Your OpenAI API Key

To use OpenAI models in BotGem, you'll need to obtain an API key:

1. Visit the [OpenAI Platform](https://platform.openai.com/api-keys) and sign in to your account
2. Click on "Create new secret key"
3. Give your key a name (optional) and click "Create secret key"
4. Copy the API key immediately (you won't be able to see it again)

## Configuring OpenAI in BotGem

Once you have your API key, follow these steps to configure OpenAI in BotGem:

1. Open BotGem and go to **Setting**
2. Navigate to the **Service Provider** section
3. Find the **OpenAI** provider in the list
4. Enter your OpenAI API key in the designated field
5. Click **Save** to apply your changes
